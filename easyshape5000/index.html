Welcome to EasyShape5000

Screen too narrow. If possible, rotate your device. UI is optimized for small screens but only in a horizontal orientation.

EasyShape5000 is a CAM tool for CNC routing.

Initial public release. While its foundations are shared with EasyTrace5000, more testing is needed. Please report unexpected behavior.

Toolpath Optimizations were also fully reworked so double check output (ncviewer).
